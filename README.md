## HOW TO RUN THE APPLICATION

Make sure you have `java` and `maven` installed. Run the following commands:

- `mvn clean install`
- `java -jar target/TampereTrafficApp-0.0.1-SNAPSHOT.jar`
- open http://localhost:8080/
